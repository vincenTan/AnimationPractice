This is my second Unity project. This game was made by referencing the following video:
https://www.youtube.com/watch?v=wdOk5QXYC6Y&t=21s

Controls:
wasd or arrowkeys to move
(cannot move sideways, only tilt direction)
hold shift and movekey to run
ctrl(left) + 1,2,3,4 to dance (spammable, like in League of Legends)
spacebar to jump
